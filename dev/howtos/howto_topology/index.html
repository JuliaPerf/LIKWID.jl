How CPU / NUMA Topology

The basis functionality of likwid-topology.

CPU

Query CPU topology information:

using LIKWID # hide
topo = LIKWID.get_cpu_topology()
topo.threadPool
topo.cacheLevels
3-element Vector{LIKWID.CacheLevel}:
 LIKWID.CacheLevel(0, :data, 8, 64, 64, 32768, 1, 0)
 LIKWID.CacheLevel(1, :unified, 8, 1024, 64, 524288, 1, 0)
 LIKWID.CacheLevel(2, :unified, 16, 32768, 64, 33554432, 8, 0)

LIKWID.print_cpu_topology()
