Like I Knew What I am Doing

LIKWID.jl is a Julia wrapper for LIKWID.

Installation

Prerequisites: You must have likwid installed (see the build & install instructions),

LIKWID.jl is a registered Julia package. Hence, you can simply add it to your Julia environment with the command

] add LIKWID
